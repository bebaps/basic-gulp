# Basic Gulp Setup

## Installation
Clone or fork the repo on [Github](https://github.com/bebaps/basic-gulp).

```bash
git clone git@github.com:bebaps/basic-gulp.git
```

## Usage
Install the dependancies via

```bash
npm i
```

You can then start a development server via

```bash
gulp
```

## Dev Dependencies
- [Gulp](https://gulpjs.com/)
- [PostCSS](https://postcss.org/)

## Contributing
Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.
